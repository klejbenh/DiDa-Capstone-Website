Backend Connection Plan

How this frontend will connect to the music-analysis backend

Step 1 Spotify login

Users log in through Spotify so the platform can request permission to read playlist and track information tied to their account.

Step 2 Playlist data retrieval

Once the user signs in, Spotify provides playlist and track information through its API. The frontend collects the selected playlist ID, track IDs, and related metadata, then sends that structured data to the backend through an API request so the analysis pipeline can begin.

Step 3 Backend analysis

The backend uses the playlist data as input for feature extraction, transition scoring, and sequencing logic to analyze how songs should flow together.

Step 4 Generated sequence

The optimized song order is returned to the frontend, where users can view a smarter playlist built from their Spotify music data.
