How To Use The Website

Start here before running the playlist analysis.

To use this project, you should go to the Analysis page. That page is where the sample backend analysis is shown, including how playlist data would be processed, scored, and reordered.

Step 1 Log in with Spotify

Use the Spotify login button at the top of the page so the system can access playlist and track data connected to your account.

Step 2 Go to Analysis

Click the Analysis page in the top navigation bar. This is the page users must visit to run or view the playlist analysis workflow.

Step 3 Run the analysis

The backend extracts audio features, scores possible transitions, and builds an optimized song order based on flow and compatibility.

Step 4 Review the sequence

Review the returned playlist order and use the other pages to understand the pipeline, modules, users, backend connection, and team focus.
