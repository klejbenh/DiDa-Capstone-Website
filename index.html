How To Use The Tool

Step-by-step instructions

Step 1 Log in with Spotify

Use the Spotify login button at the top of the page so the system can access playlist and track data connected to your account.

Step 2 Select a playlist

Choose the playlist you want analyzed so the frontend can send the relevant track IDs and metadata to the backend.

Step 3 Run the analysis

The backend extracts audio features, scores possible transitions, and builds an optimized song order based on flow and compatibility.

Step 4 Review the generated sequence

Look through the returned playlist order and use the other pages to understand the pipeline, modules, sample output, user profiles, and team focus behind the result.
